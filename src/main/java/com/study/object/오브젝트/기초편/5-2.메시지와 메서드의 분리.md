### 메시지와 메서드
* 메시지 : 객체 사이의 의사소통 수단
  * 다른 각체에게 책임 수행을 요청하는 커뮤니케이션 수단
* 메서드 : 메시지를 처리하는 방법
  * 메시지를 수신한 타입별로 책임을 수행하는 방식

### 메시지와 메서드의 분리는 다형성의 기반
* 다형성은 메시지와 메서드를 분리하고 런타임에 이들을 조합해서 다양한 종류의 객체들이 협력할 수 있도록 구성하는 객체지향만의 독특한 메커니즘에 기반하고 있다.
